# Credit Scoring Model (Probability of Default)

Пет-проект по разработке и оптимизации модели кредитного скоринга на данных Kaggle "Give Me Some Credit".

## Цель проекта

Предсказать вероятность кредитного дефолта (Probability of Default, PD) на горизонте 2 лет на основе исторических данных.

## Описание

Задача бинарной классификации: `target = 1` дефолт / `0` не дефолт.
